Flow Control

流程控制与错误处理

 
条件判断语句
if...else
if (condition_1) { statement_1; } else if (condition_2) { statement_2; } else if (condition_n_1) { statement_n; } else { statement_last; }
switch
switch (expression) { case label_1: statements_1 [break;] case label_2: statements_2 [break;] ... default: statements_def [break;] }
 
异常处理语句
throw
throw true;
try...catch
try { monthName = 123; } catch (e) { monthName = 0; }
finally
try { monthName = 123; } catch (e) { monthName = 0; } finally { monthName = 0; }
