数据类型转换

 
JS是动态类型
 
var 变量 = "变量";
变量 = 123;
变量的值是多少？
text
 
字符串+数字，会把字符串和数字连接起来
变量 = 变量 + "456";
变量的值是多少？
text
但其他运算符不会这样
变量 = 变量 - "456";
变量的值是多少？
text
